# CVCraft
CVCraft is a clean and responsive resume builder that allows users to generate beautiful resumes instantly and export them as PDF.
# ✨ CVCraft

CVCraft is a responsive Resume Builder web app made using HTML, CSS and JavaScript.

Users can:
- Create resumes instantly
- Upload profile images
- Preview resumes live
- Download resumes as PDF
- Use the app on both mobile and desktop
